Maybe we should look into this:

http://www.bebits.com/app/698

License is "Public Domain" and in the changelog it mentions:

About International Keymap Pack:
Definitive keymap collection for various national keyboard layouts

These keymaps are designed based on following common policies:

- Based on Windows 98/2000 and PC DOS.
- AltGr works as Option key, i.e. DOS/Windows-like.
(yet Vulcan Deach Grip compatible)
- Keep clean and concise. (no redundant characters in Control,
Option and CapsLock maps)
- Euro enabled. (generally assigned to AltGr+E)
- Keypad decimal point produces either period or comma according to the 
locale.
- Ctrl+Space produces 0x00 for Emacs users.

Common problems seen in BeOS standard keymaps are fixed as follows:

- The 102nd key (next to left Shift) activated.
- CapsLock map fixed.
- Numeric keypad layout and Enter key fixed.
- Dead key combinations fixed.

-Bruno

Ps: The Brazilian(ABNT2) layout we use is from this package and it works 
almost perfectly. The exception is that under Haiku ~ (which is a dead 
key) + SPACE does not produce ~ (it produces nothing) and it should. But 
the same keymap works on Zeta.
