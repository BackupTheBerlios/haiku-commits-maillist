The buffer size is too large causing ethernet_receive_data() to fail 
with "scattered I/O is not yet supported by ethernet device". That's in 
network/devices/ethernet/ethernet.cpp. Really the 2000 was just an 
arbitrary value > 1500. Using 1500 should be fine as that's what normal 
ethernet frames can become in size anyway.
