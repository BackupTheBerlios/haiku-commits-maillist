Michael Pfeiffer <michael.w.pfeiffer at gmail.com> wrote:
> > -		delete buffer;
> > +		delete[] buffer;
> Just curious, is deleting the array this way really required? I mean  
> it's an array of a simple type and I don't even know if there exist a  
> destructor for that.

For simple types it shouldn't be any actual difference, even though 
it's certainly clearer to do it. Not using the delete[] will usually 
only cause the destructors not to be called (as the allocator usually 
maintains the size itself).
