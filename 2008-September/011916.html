I know it is already too late, but I am against this rule. I would  
favor always to use parenthesis:
1) It is unsymmetrical
    if (...) {
       more than
       a single line
    } else
       a single line only

2) When you write code in order to place the "{" you have to know how  
many lines you are going to write inside the block. Modern editors can  
put them automatically after "if ()" or "else".

3) Whenever you change code inside a block you have to update the  
parenthesis if it changes from multi line to single line or vice versa.
