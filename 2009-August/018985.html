So can we for BFS.
Tested in ZETA:
setmime -add application/x-bfs-image -short "BFS Image" -long "Be File 
System Image" -extension bfs -sniffRule '0.60          
([544]"1SFBEGIB")'

for big endian too it should be:
setmime -add application/x-bfs-image -short "BFS Image" -long "Be File 
System Image" -extension bfs -sniffRule '0.60          [544]("1SFBEGIB"
|"1SFBBIGE")'
