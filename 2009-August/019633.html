Do we have to use the "auto" prefix?

The reason for the "auto" in "autostart" is to make a
distinction between items in the menu that you start
manually (most of the menu) and the items that the system
starts automatically (the "Autostart" folder).

If the folder is not to be shown in Deskbar, there is
no need for that distinction in its name. The "auto"
is redundant.

The system does all kinds of things automatically.
No need to be all explicit about it.
