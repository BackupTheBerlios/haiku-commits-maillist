You most certainly won't get the raw device inside a VM. The keyboard 
looks pretty much exactly like the one Axel had at his place while I 
completed usb_hid. It was actually the first device I tested it on and 
everything except for the favorite keys should trigger an unmapped key. 
The zoom slider is actually implemented as two keys, if you push it 
upwards it'll send one keycode, otherwise it'll send the other. You 
should get both as unmapped keys with their HID usage code. See http://www.usb.org/developers/devclass_docs/Hut1_11.pdf 
for reference of these codes.
