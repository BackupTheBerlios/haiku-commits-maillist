Here you actually have two grid layouts (the base one with only one
child in the grid, that child being the second grid layout from the
builder.) It probably doesn't matter that much or hurt anything
though. Generally the layout you set with SetLayout is sort of
throwaway, especially if you then use a builder. Generally we use some
sort of BGroupLayout for that.

It might be nice if you could use the builders with a layout you have
already set. I've never tried it, so maybe it is possible...
