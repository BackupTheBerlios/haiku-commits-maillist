It sounds like you just need to set an explicit horizontal minimum size of 
some random value which looks good, and an explicit maximum width of 
B_SIZE_UNLIMITED. What probably happens is that the string view resets 
these when the string changes, but you want to allow the situation where 
the string does not fit, or where the view is larger than the string.
