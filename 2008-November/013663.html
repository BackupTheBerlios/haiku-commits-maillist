Yes. Use the new layout system. See src/preferences/appearance/ for how to work with BTabViews in layouted mode. This is actually a defficiency in the BeOS implementation as well. The views which are not visible actually get detached from the window, so they don't receive frame events anymore. To work around that, you could also override AttachedToWindow() and position yourself within your Parent(). But please use the new layout management instead. :-)
