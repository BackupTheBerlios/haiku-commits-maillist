Why? Unless this is just not functional in Haiku, there is a very real
and legitimate reason this existed. Most 'normal' users - which is
what you appear to be trying to warp the interface for - only have one
BFS partition. This setting turned on means that if they put in a BFS
CD, such as a disc for a commercial application, it can place an
Installer icon on the desktop, or similar.

Its far, far more likely to be a problem for developers than any
hypothetical 'normal' users we'll be getting in the future, but anyone
who's irritated -knows how to turn it off-.
