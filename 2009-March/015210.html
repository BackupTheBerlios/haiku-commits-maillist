I didn't really consider clean GCC4 images, I thought everyone would use

in using GCC2 Pe, or is there? Now, WonderBrush would be able to use 
Translaters, which is somewhat useful. But like I said, it's much more 
work... which I need to do one day anyways... so I might as well start now. 
Although I was hoping to do an entirely new WonderBrush with a new backend
