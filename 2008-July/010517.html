Hey,

on http://maurice.kaldience.com/?p=77 you can see a small tutorial of 
how to use Cygwin to successfully build Haiku on Windows. It's pretty 
much the same like the Ubuntu tutorial.
There is one patch missing for jam, which is including unistd.h in any case.

The problem is that we cannot use OS_CYGWIN/__cygwin__ for reasons 
described in one of the previous commits (.exe extensions). But we 
definitely need unistd.h as otherwise lseek will fail and this causes 
recompilation of each library for each call of jam.

As we are using our own jam already, I am unsure if we could just add 
this line to jam in case it does not break on any other platform. We 
haven't had the time to try it out on linux/r5/etc. so we waited for 
your feedback on this one.

Regards,

Maurice
