It's about using the ACPI module to get the PCI interrupt mapping from PIRQs
to global interrupt numbers. This is required to use IOAPICs, because without
this mapping it can't be configured fully and PCI devices don't get any
interrupts. In the end ACPI could also be used to do the remapping. That
should be put into the PCI bus_manager though.
