BTW, the typical name for an ID getter would be ID(). Getters are prefixed with "Get" only when (a) the method name would otherwise clash with a type name, or (b) the value is returned through a reference/pointer parameter. Furthermore the "void" parameter is redundant in C++, and getters should be "const", if possible.
