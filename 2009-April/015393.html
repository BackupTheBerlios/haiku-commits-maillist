Please note that you'll have to run makebootable in case you are 
updating an existing installation before you can remove the /boot/beos 
directory, since it will still boot from that one.
To do so from the build host you can do something like:
$ jam -q run :/<build\>makebootable <path-to-my-image>

In case you are using VMware images only, use:
$ jam -q run :/<build\>makebootable --start-offset 65536 <path-to-my-
vmware-image>
