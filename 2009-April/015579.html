ATM we have neither network remote GUI support nor multiple platforms 
supporting it. Besides I would be in favor of implementing that 
transparently to the application in that it would always talk to the local 
app server with the app server using a special protocol forwarding the 
connection to the remote app server. This way it would even be possible to 
migrate the application GUI transparently to another machine 
(home<->phone<->work).
