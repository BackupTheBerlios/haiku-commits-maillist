Axel Dörfler wrote:
> 
> That sounds like a good idea, too. Dunno how to do this best, though, 
> maybe some awk/sed trickery with sysinfo.

mem_free=`sysinfo | grep "bytes free" | cut -d " " -f 1`

echo Free memory: $mem_free
if [ $mem_free -gt 30000000 ]; then
	echo "Free memory is greater than 30000000"
fi

Replace 30000000 with whatever number makes sense.
