The recommended way is to create your personal build/jam/UserBuildConfig 
file and use it to configure the build to your needs. Among other things 
you can turn on/off debugging for whole source trees, single directories, 
or even individual object files. E.g. turn on debugging for the kernel via:

  SetConfigVar DEBUG : HAIKU_TOP src system kernel : 1 : global ;

When you build the disk image exactly what is specified there at that time 
will end up on the image. No need for manual copying.

For more insights what can be done in the UserBuildConfig have a look at 
build/jam/UserBuildConfig.sample
