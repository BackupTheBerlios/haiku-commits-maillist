Btw, while I didn't find any provision in the PS/2 protocol to pass the 
keyboard language, the USB HID has a way to give the localization of 
the device:

"The value bCountryCode identifies which country the hardware is 
localized
for. Most hardware is not localized and thus this value would be zero 
(0).
However, keyboards may use the field to indicate the language of the 
key caps.
Devices are not required to place a value other than zero in this 
field, but some operating environments may require this information. 
The following 
table specifies the valid country codes."

p32, http://www.usb.org/developers/devclass_docs/HID1_11.pdf

While people can still reorder caps (dvorak...) it might be nice to use 
that if present to select the default keymap.
