> Also, it does not seem to work correctly with regards to modifiers. How
> do I select which modifiers I need to have pressed to make the dead key
> work? For example, in the American keymap, the '`' key is available via
> the "option" modifier as dead key, but is a standard key when you don't
> press that modifier.

Yes, currently these two things are separated: the menu only defines which 
characters work as dead key triggers. You can enable/disable the dead key / 
modifier combination by clicking on the respective key with the middle mouse 
button once you have pressed the required modifiers.
So the menu sets up dead_key[1] and the middle clicks add/remove modifiers 
from the dead_key_table field. 
But maybe that feature is too complicated/hidden?

> I am not sure if that would be the perfect solution, but we could make

Right, we could do that, too. This would probably be more obvious than having 
to use the middle mouse button.
