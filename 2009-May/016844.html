It was my pleasure. At first I feared I wouldn't understand how these 
things all fit together, but it's indeed very nicely structured and 
readable. So it wasn't too much of an effort to make this happen :-)

The nice thing is that you can now handle icons in a more automated 
fashion using the standard system APIs. You could for example set the 
desired output size in the translator preferences, navigate to a folder 
full of icons (like "src/data/artwork/icons") and then use a one-liner 
like:

for FILE in *; do translate $FILE export-path/$FILE.png 'PNG '; done

To export them all in one go to PNG files in some folder. I could think 
of a few situations where this might come in handy.
